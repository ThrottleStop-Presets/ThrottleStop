## What's Included
* 📊 Ready-made ThrottleStop profiles and presets for different CPU generations (Intel 12th–15th gen, Core Ultra)
* 📈 Detailed performance testing templates, temperature logging and benchmark comparison sheets
* 🔍 Hardware-specific checklists and recommended settings for popular laptop and desktop models
* 🛠️ Custom undervolting configurations, speed shift, power limit and turbo boost optimization profiles
* 📸 High-quality screenshots, step-by-step setup guides and before/after performance comparisons

## How to use
1. Download the archive from the **Releases** tab on the right → select the desired .zip file
2. Extract the files to any convenient folder
3. Run ThrottleStop and import the desired profile
4. Apply settings according to your CPU model and cooling system
5. Test performance and temperatures under gaming or heavy load
6. Fine-tune the parameters based on your specific hardware and thermal limits

## Important Notes
- All presets are provided as is — always monitor CPU temperatures and stability during initial testing
- Create a system restore point before applying any changes
- Results depend heavily on your cooling solution, case airflow and silicon quality
- Compatible with Windows 10 and Windows 11 (2026 builds)
